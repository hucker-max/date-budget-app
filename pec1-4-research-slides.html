PEC1-4 / 光電融合 / 材料課題

PEC1-4のリサーチ結果と材料課題

目的は、光電融合が「どこまでコンピュータに近づく技術なのか」を段階ごとに整理し、 各段階で何を実現したいのか、材料面ではどこが難しそうかを簡単に俯瞰することです。

今回の見方

PEC-1からPEC-4へ進むほど、光はネットワークから装置内部、パッケージ近傍、チップ内部へ近づきます。 そのぶん、通信理論よりも材料・実装・熱・信頼性の問題が強く効いてきます。

PEC-1

ネットワークの光化。遠くを低損失・大容量でつなぐ段階。

PEC-2

ボード間の光化。装置内通信の電力を下げたい段階。

PEC-3

パッケージ間の光化。材料と実装が主戦場になりやすい段階。

PEC-4

ダイ間・チップ内の光化。将来像が強く、接合と熱が鍵になりそうな段階。

卒業研究向けの簡易まとめ 1 / 6
PEC-1 / PEC-2

外のネットワークから装置内部へ光を入れる

PEC-1で技術としてやりたいこと

データセンター間・サーバ間の大容量通信を、低消費電力・低遅延で成立させたい。

石英ファイバ シリカ PLC / AWG Si フォトニクス InP 系
  • 長距離での伝送損失を抑える
  • 波長分離と多重伝送を安定に行う
  • 耐湿・耐熱・長期安定性が重要

PEC-2で技術としてやりたいこと

装置内のボード間配線を光化し、Switch ASIC近傍まで optical engine を寄せて通信電力を下げたい。

Si / SiO2 / InP SiOx 導波路 接着材 コネクタ材
  • 近接実装で熱と接合が効いてくる
  • アライメントとコネクタ小型化が課題
  • 通信材料に加えて実装材料が重要になる

簡単に言うと

PEC-1は「光通信材料」が主役で、PEC-2は「光通信材料 + 実装材料」が主役です。 ここから光電融合は、通信技術だけでなく計算機実装の技術になっていきます。

PEC-1は遠距離、PEC-2は装置内部 2 / 6
PEC-3

パッケージ間を光でつなぎたいが、材料課題が一気に重くなる

技術としてやりたいこと

  • CPU / GPU などのパッケージ間通信を光化したい
  • 短距離・高密度・低消費電力の接続を成立させたい
  • Si フォトニクスと膜型 III-V を近接統合した光 I/O を実装したい

材料的な課題

  • 熱で光結合条件や波長が変わる
  • CTE差と反りでアライメントがずれる
  • 1310 / 1550nm 帯の損失が導波性能に効く
  • リフロー、熱サイクル、剥離、量産性まで同時に見る必要がある
層の見方 Siフォトニクス光回路 / InP系膜型活性層 / 低屈折率クラッド / 放熱層 / 接着・封止材
研究の軸 熱・反り・光損失・信頼性・量産性を同時に扱うこと
印象 PEC-3は、光電融合の中でも最も「材料・実装・評価」に落とし込みやすい段階
卒研テーマとして一番自然に切り出しやすい領域 3 / 6
PEC-4

ダイ間・チップ内まで光を持ち込みたいが、仮説色が強い

技術としてやりたいこと

  • die-to-die や chip内 optical layer を成立させたい
  • チップ内部の信号移動そのものを光へ寄せたい
  • 将来的には core-to-core optical transmission まで視野に入る

材料的な課題

  • bonding の位置ずれ・接合信頼性がすぐ性能に効く
  • optical layer と electrical layer の熱干渉が起きやすい
  • 応力、封止、長期信頼性まで含めた積層設計が必要
  • 実装像はまだ仮説が多く、今後の検証が必要

想定材料

Si / SiN 導波路、InP 薄膜 or Ge / Si、低k / クラッド、Cu / SiCN bonding、熱拡散材。

強く効きそうな課題

導波路そのものより、die-to-die の bonding 層と熱マネジメントが律速になる可能性があります。

いまの見立て

全面光化より、まずは I/O タイルや chiplet 境界から部分光化される方が自然そうです。

PEC-4は公式 + 妥当な実装仮説として見る段階 4 / 6
MGC材料

MGC材料に引きつけると、役割ごとに見方が変わる

OPE

高温耐性と CTE の公開エビデンスは比較的強いです。

高温耐性 CTE 近赤外損失不明
  • 高温実装近傍の電気系材料としては有望
  • 光導波路材料としては判断材料不足

BT樹脂系

基板材料としては自然に有望ですが、PEC-3向け光学評価が弱いです。

基板材料 反り評価不足 光学損失不明
  • パッケージ基材・電気配線側に向いていそう
  • 260℃ピークや剥離データがほしい

ユピゼータEP

光学樹脂・光学部材候補としては面白いですが、PEC-3向け定量データが不足しています。

光学樹脂 耐熱不足 CTE不明
  • レンズ・光学部材寄りに見る方が自然
  • 近赤外・耐熱・寸法安定性の数値が必要

簡単に言うと

3材料ともそのまま PEC-3 に使えるとはまだ言い切れません。 ただし、OPE は実装材料寄り、BT は基板材料寄り、ユピゼータEP は光学部材寄り、という見方はしやすくなっています。

1材料で全部を担うより、役割別に見る方が整理しやすい 5 / 6
まとめ

研究としては「どの材料がどの役割を担えるか」を見極める話

技術として何をしたいのか

光を、ネットワークから装置内部、パッケージ間、最終的にはダイ間・チップ内へ近づけて、 データ移動の電力と帯域の問題を解きたい。

材料研究としての問い

  • PEC-3 では何を満たせば材料候補と言えるのか
  • どの材料が基板材・導波路材・光学部材・接着材のどこに向くのか
  • 何のデータが足りないと最終判断できないのか
重要評価軸 260℃リフロー耐性 / 1310・1550nm 光学損失 / CTE・寸法安定性
次にやること 公開エビデンス比較、評価項目と測定方法の対応付け、MGCへの質問設計
研究の着地点 光電融合を成立させる材料条件を整理し、候補材料がどの役割なら使えそうかを見極めること
